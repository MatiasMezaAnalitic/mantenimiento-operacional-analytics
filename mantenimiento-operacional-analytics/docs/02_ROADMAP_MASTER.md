# ROADMAP INDUSTRIAL MAESTRO

## FUNCIÓN DEL DOCUMENTO

Este documento define:

* dirección estratégica
* secuencia de aprendizaje
* dependencias entre habilidades
* proyectos obligatorios
* hitos esperados

NO define:

* progreso real
* habilidades dominadas
* entregables completados
* estado actual del usuario

La fuente de verdad para progreso es:

1. STATE_FILE
2. KNOWLEDGE_BASE

Si existe conflicto:

STATE_FILE > KNOWLEDGE_BASE > ROADMAP_MASTER

---

# PRINCIPIO DE PROGRESIÓN

El roadmap define la ruta ideal.

No implica avance automático.

No se puede avanzar de fase únicamente por tiempo transcurrido.

Para avanzar se requiere:

* criterio de salida cumplido
* entregable validado
* evidencia generada
* actualización en STATE_FILE

# FILTRO DE RETORNO LABORAL

Toda actividad propuesta debe responder afirmativamente:

* ¿Me acerca a minería o industria?
* ¿Me vuelve más analítico?
* ¿Me genera evidencia profesional?
* ¿Me ayuda a automatizar?
* ¿Me mejora empleabilidad?

Si la respuesta es NO:

La actividad debe eliminarse.
