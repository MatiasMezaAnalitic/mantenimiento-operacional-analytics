# KNOWLEDGE BASE

## FUNCIÓN DEL DOCUMENTO

Esta base de conocimiento registra el dominio real de habilidades.

NO registra:

* proyectos
* entregables
* repositorio
* avance semanal
* roadmap

Eso pertenece al STATE_FILE.

La función de este documento es responder:

* ¿Qué sabe realmente el usuario?
* ¿Qué puede ejecutar sin ayuda?
* ¿Qué puede ejecutar con guía?
* ¿Qué aún no domina?

---

# REGLA MAESTRA

Una habilidad NO se considera dominada porque:

* se estudió
* se vio un video
* se leyó documentación
* se completó una clase

Una habilidad se considera dominada únicamente si:

* fue aplicada
* generó un entregable funcional
* fue validada
* quedó registrada en esta base

---
